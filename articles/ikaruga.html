IKARUGA

白と黒が織りなす、シューティングゲームの詩

// 2001 — Treasure DC / GC / Switch / PS4 / Steam 1〜2人プレイ

概要 — 弾幕を「解く」縦スクロールシューティング

斑鳩は、トレジャーが開発した縦スクロールシューティングゲームだ。2001年にナムコのアーケードとして稼働を開始し、同年にDreamcastへ移植。2003年にはNintendo GameCubeにも移植され、2014年にはSteamでPC版が配信。2018年にはNintendo SwitchとPlayStation 4にも対応し、現在でも現行機で遊べる。

本作の核心は「ポラリティ（属性）切り替え」システムだ。自機は白と黒の2属性を切り替えることができ、同属性の弾を吸収してチャージゲージを溜め、異属性の敵に2倍ダメージを与えることができる。弾幕を「避ける」のではなく「吸収と回避を使い分ける」——このシステムが、斑鳩をSTGの歴史に唯一無二の存在として刻んだ。

ゲーム全5章。一見シンプルに見えるルールが、章を重ねるごとに深みを増し、最終的には「すべての弾幕がパズルだった」と気づかされる。

// IKARUGA — Official Trailer (Treasure)

ゲームプレイ — 吸収と破壊、陰陽の連鎖

操作は非常にシンプルだ。移動・ショット・属性切り替えの3アクションのみ。しかしこの3つの組み合わせで生まれる戦略の深さは底知れない。

敵は白と黒の2種類に分類され、それぞれ対応した属性の弾を放つ。同属性の弾を5発吸収するとホーミングレーザーを発射でき、敵を一気に殲滅できる。しかし吸収に頼りすぎると、素のショット火力が落ちる。吸収と回避のバランスをリアルタイムで判断し続けることが、高スコアへの道だ。

チェーンコンボは本作最大の特徴のひとつ。同属性の敵3機を連続撃破するたびにチェーンカウントが増加し、3連鎖×8セット（合計72機）でパーフェクトチェーン達成となる。これを全章でノーミス達成する「絶対ノーミスパーフェクトチェーン」は、世界中のプレイヤーが今なお挑戦し続ける至高の課題だ。

斑鳩は、シューティングゲームというジャンルを超えて「ゲームデザインとは何か」を教えてくれる

— ゲームデザイナー・岩井俊雄氏

ビジュアルと世界観 — 和の静謐と機械の暴力

斑鳩のビジュアルは白と黒を基調とした禅的なデザインで統一されている。鳥の名を冠したステージ、仏教・神道を想起させる建築物、無機質な機械兵器と精神的な世界観の融合——これがプレイヤーに「美しい戦場」の体験を与える。

同じくトレジャーが開発したラジアントシルバーガン（1998年）の精神的後継作として位置づけられており、世界観を共有する部分もある。ラジアントシルバーガンのファンにとって、斑鳩は「さらに洗練されたトレジャーの答え」だ。

2人協力プレイの変態的な深み

斑鳩は2人同時プレイに対応しているが、これが本作のカオスをさらに加速させる。2人の自機が異なる属性を担当し、連携しながら弾幕を処理する——理論上は最強の布陣だが、互いの動きを把握しながら属性を合わせ続けることは至難の業だ。2人プレイの動画はYouTubeでも多数上がっており、それ自体がエンターテインメントと化している。
